# Dinão Skull Thermo - Landing Page

Projeto de alta performance criado para o suplemento Dinão Skull Thermo.

## Como Iniciar

1. Certifique-se de estar na pasta do projeto: `D:\dinao-skull-thermo`
2. Instale as dependências (se ainda não o fez): `npm install`
3. Inicie o servidor de desenvolvimento: `npm run dev`
4. Acesse: `http://localhost:3000`

## Funcionalidades

- **Landing Page principal**: `/` (Altamente otimizada para conversão)
- **Modo Apresentação**: `/apresentacao` (Slides interativos para vendas)
- **Painel Administrativo**: `/admin` (Para editar textos e links - Senha: `dinao2026`)

## Estrutura de Cores (Brand)

- Primária: `#FF0000` (Vermelho Dinão)
- Fundo: `#000000` (Preto Industrial)
- Detalhes: `#1a1a1a` / `#2d2d2d`

## Imagens Necessárias

Coloque os seguintes arquivos na pasta `public/`:
- `pote_preto.png` (Produto principal)
- `og-image.png` (Para redes sociais)
- `favicon.ico`
